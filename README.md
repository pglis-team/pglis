## The PgLis model 

a tool for the long-term forecast of GCR in the heliosphere

- a python implementation using pre-computed tables available at https://doi.org/10.5281/zenodo.19607311 

# Functionality
 - using solar proxy value (SSN) from NOAA SPACE WEATHER PREDICTION CENTER (https://www.swpc.noaa.gov/products/solar-cycle-progression)

 - forecast GCR nuclei from proton to Nickel Z=1-28
